Pagina Principal

Esta pagina contiene la información general sobre el OVI, sus objetivos, los objetivos del curso y las temáticas a las cuales hace referencia el OVI.

Información general de OVI

Se construirá el OVI con el fin de cumplir con los objetivos planteados mediante herramientas informáticas de diseños web.

Objetivos del OVI

Dar como resultado una pagina web la cual sea efectiva y asertiva para el desarrollo satisfactorio de los objetivos planteados

Objetivos del curso

Fundamentar teóricamente al estudiante sobre las generalidades básicas del Diseño Web

Temáticas referentes

Las temáticas referentes son: de diseño web como HTML5 Y CSS3, y con herramientas de control de versiones como GIT Y GITHUB.
